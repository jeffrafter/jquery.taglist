jquery.taglist by jeffrafter

Simple tag list editor that converts the tags to blocks as you type

Example

Dependencies

JQuery 1.4+

Basic Usage

  <div class="container">
    <div class="taglist"></div>
  </div> 
 
  <script type="text/javascript">
    $('div.taglist').taglist(['The first tag', 'The second tag', 'Third tags a charm', 'Even more tags']);
  </script>
